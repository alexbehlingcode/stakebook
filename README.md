# Stakebook — Smart Betting Tracker

The personal betting journal that keeps you honest. Track bets, analyze performance, and bet responsibly.

## Quick Start

```bash
npm install
npm run dev
# Open http://localhost:5173
```

## Deploy to Vercel

1. Push to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Initial commit — Stakebook v1.0"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/stakebook.git
   git push -u origin main
   ```

2. Go to [vercel.com](https://vercel.com) → Sign in with GitHub → Add New Project → Import `stakebook` → Deploy

3. Optional: Add custom domain (stakebook.io) in Vercel Settings → Domains

## Tech Stack
- React 18 + Recharts + Vite
- Hosted on Vercel (free tier)
- PWA-ready for mobile install
